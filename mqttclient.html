Privacy Policy for MQTT Client & UDP Bridge

This application operates as a network utility tool on your device.

1. Data Collection

The developer does NOT collect, store, or transmit any personal user data or usage analytics to external servers.

All MQTT and UDP traffic is processed strictly in RAM and routed solely between the endpoints configured by the user (the MQTT Broker and UDP peers). Messages are logged locally on the screen for debugging purposes only.

2. Permissions

  • Network / Internet: Used solely to establish TCP connections to the user-defined MQTT Broker and to listen for/send UDP packets on the local network.
  • Foreground Service: Used to keep the network sockets active and the bridging logic running when the application is minimized or in the background.
  • Wake Lock: Used to prevent the Android Operating System from putting the CPU to sleep, which would terminate active network connections.

3. Logs

Data logs displayed within the application interface are temporary, stored only in volatile memory, and are permanently deleted when the application is closed.
